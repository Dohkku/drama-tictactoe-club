# Sistema 4: Layout Manager

Gestiona cómo se distribuyen los paneles en pantalla. Transiciones entre modos (fullscreen cinematic, split, board only). Highlights de panel activo.

## Responsabilidades

- Tres modos de layout: fullscreen (solo cinemática), split (ambos), board_only (solo tablero)
- Transiciones animadas entre modos (tween de stretch ratios)
- Adaptación portrait/landscape
- Panel highlights (glow en el panel que requiere atención)
- Separador visual entre paneles

## NO es responsable de

- Contenido de los paneles (→ Cinematic / Board Visuals)
- Cuándo cambiar de layout (→ Scene Runner / Match Orchestrator)

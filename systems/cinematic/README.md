# Sistema 3: Cinematic Stage

Todo lo visual narrativo: personajes en escena, cámara virtual, efectos, diálogos, title cards. No sabe que existe un tablero.

## Responsabilidades

- Registro y display de personajes (CharacterSlot)
- Posicionamiento en escena (far_left → far_right, 7 posiciones)
- Entradas/salidas animadas con dirección
- Expresiones, poses, dirección de mirada
- Cámara virtual (close-up, pull-back, snap, reset, modos smooth/snappy)
- Efectos (shake, flash, speed lines)
- Caja de diálogo con typewriter, voces procedurales, opciones
- Title cards con fade in/out
- Fondos (color, textura, video)

## NO es responsable de

- Tablero de juego (→ Board Visuals)
- Lógica de partidas (→ Board Logic)
- Flujo narrativo / scripts (→ Scene Runner)
- Layout de paneles (→ Layout Manager)
